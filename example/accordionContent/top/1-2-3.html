In the case where an item has a bottomContent and a subcollection, the content is going to be positioned bellow the collection.

Both the topContent and the bottomContent attributes are optional in each item an can coexist, meaning you can expand an item thas has a collection of items of its own, but can also have at the same time information on top and below this items.

For better organizing, if there is no need to add information bellow a subCollection, it's recommended to use only the topContent.
